"#t01 th" : 헤더 셀이 있는 행의 속성을 지정

#t01 th { background-color: black; color: white; } ==> t01테이블의 제목 행 배경색을 검정, 글 색을 흰색으로 지정


"#t01 tr:nth-child(even)" : 짝수번째 행의 속성을 지정

#t01 tr:nth-child(even) { background-color: red; } ==> t01테이블의 짝수번째 행의 배경 색을 빨간색으로 지정


"#t01 tr:nth-child(odd)" : 홀수번째 행의 속성을 지정

#t01 tr:nth-child(odd) { background-color: blue; } ==> t01테이블의 홀수번째 행의 배경 색을 파란색으로 지정
